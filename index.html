Status Colors
Calibrated success, danger, warning, and info indicators. Body text auto-flips to #1A1A1A or #FFFFFF based on background luminance, so each color is guaranteed readable without manual contrast tuning.
SECTION 04
Announcement Colors
Preview of leveled announcements. Each row can be set to Tinted (status hex at 20% opacity, dark body, primary link) or Full (status hex at 100%, body auto-flips to dark or white based on background luminance). On a light Full background the link tries Primary first and auto-flips to dark when Primary fails WCAG AA against that background. Add as many as the client uses.
SECTION 05
Typography
Secondary font drives all headings (H1-H6) by default; primary font is locked for body and small text. Use the dropdown on any heading row to override that row to primary. Base size is 18px; the heading scale steps up by 1.25x, 1.5x, 1.75x, 2x, and 2.5x.
